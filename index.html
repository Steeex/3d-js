3D-JS is an attempt to write a software renderer in JS and using it to build a Minecraft like "game". 3D-JS is written completely in vanilla JS, meaning no 3D libraries like Three.js or Babylon.js, nor harware acceleration (only uses the CPU). Mathematical implementations of 2D or 3D vectors and matrix are also self made and written directly in JS. It only uses the context of the canvas available in JS, trying to overcome it's poor performances.

To see a more detailed explanation, the source files and the why, checkout the GitHub repository here.
